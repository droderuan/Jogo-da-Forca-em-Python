# Jogo-da-Forca-em-Python
Jogo da Forca para o terminal feito em python 

Separado em 4 arquivos:
  Palavra: contém uma classe que serve como a classe principal que efetivamente será o jogo\n
  ValidarPalavra: contém uma classe que serve de auxilio para fazer uma validação do input\n
  Menu: contém uma classe com alguns métodos que retornam artes em ASC\n
  Main: é onde está organizada o fluxo e é o arquivo que deve ser executado\n
  
  Tentei realizar seguindo a orientação a objetos para aprimoramento desse conhecimento
